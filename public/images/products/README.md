# Product Photography

Place final brand photography here. All files: **1600 × 1200 px**, JPEG.

| File | Subject |
|------|---------|
| `corn-tortillas.jpg` | Stacked corn tortillas. No tacos or plated meals. |
| `flour-tortillas.jpg` | Stacked flour tortillas. Soft texture visible. |
| `masa.jpg` | Fresh masa in artisan presentation. Wooden surface. |
| `tamales.jpg` | Wrapped tamales with corn husks clearly visible. |
| `spices.jpg` | Traditional Mexican spices: dried chiles, clay bowls, wooden spoons. |

Once placed, update the matching key in `app/components/photos.ts` → `PRODUCT_PHOTOS`:
```ts
corn: { src: "/images/products/corn-tortillas.jpg", ... }
```
